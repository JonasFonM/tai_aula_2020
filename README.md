# Comandos para o Git

Clonar Projeto 
```
  git clone URL
```

Add Arquivos \
  ```
  git add .
```
Realizar o commit\
  ```
  git commit -m "Sua mensagem"
```
Enviar para o github\
  ```
  git push
```
Pegar atualização do repositorio\
  ```
  git pull
```
